# 3205team
Тестовое задание для компании 3205team

## О приложении
- Frontend: React.js, Axios
- Backend: Node.js, Express.js, Typescript

## Установка
### Открываем первый терминал
### `cd server`
### `npm install`
### `npm run dev`

### Открываем второй терминал
### `cd client`
### `npm install`
### `npm start`
